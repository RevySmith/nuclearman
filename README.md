Save the Power Plant is a Python-based Hangman-style game built using Tkinter and Pillow.  
Your mission is to guess the correct shutdown code before the reactor explodes.

The game includes multiple difficulty levels, an on-screen keyboard, and custom graphics for an immersive experience.



## Features

- Multiple difficulty modes:
  - Easy
  - Medium
  - Hard
  - Phrases
  - Random
- On-screen clickable keyboard
- Manual text input for guessing letters or full words
- Custom visuals for:
  - Title screen
  - Hangman stages
  - Win screen
  - Loss screen
- Persistent win/loss tracking during runtime
- Play-again functionality without restarting the app




## Technology Used

- **Python 3**
- **Tkinter** (GUI)
- **Pillow (PIL)** for handling images
- **Random** module for selecting words
